# Sysnordic Frontend

This is the Next.js frontend application for Sysnordic's customer portal.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## About Sysnordic

Sysnordic is an Oslo-based SOC as a Service provider offering:
- 24/7 Security Monitoring
- Incident Response
- Digital Forensics
- Compliance Support (NSM, NIS2, ISO/IEC 27001)

Contact: [kontakt@sysnordic.no](mailto:kontakt@sysnordic.no)
